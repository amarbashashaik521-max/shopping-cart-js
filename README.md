# 🛒 Shopping Cart

A simple Shopping Cart web application built using **HTML**, **CSS**, and **JavaScript**.

## ✨ Features

- ➕ Add shopping items
- 💰 Enter item prices
- 🗑️ Remove items individually
- 📊 Automatically updates the total price
- ⚠️ Input validation for empty fields
- 🎨 Clean and responsive user interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation)
